Instalação do Metasploit Framework

O Metasploit é uma ferramenta para testes de penetração e exploração de vulnerabilidades.

Linux (Kali/Debian/Ubuntu)

  1. Atualize os pacotes: 
    sudo apt update
  2. Instale o Metasploit Framework: 
    sudo apt install metasploit-framework
  3. Inicie o console com: 
    msfconsole

Windows

Baixe o instalador no site oficial: metasploit.com/download e siga as instruções.
