Instalação do OpenVAS (Greenbone Vulnerability Manager)

OpenVAS é um scanner de vulnerabilidades de código aberto.

Linux (Debian/Ubuntu)

sudo apt update
sudo apt install openvas
sudo gvm-setup
sudo gvm-check-setup

Após a instalação, acesse a interface web para usar o OpenVAS.
