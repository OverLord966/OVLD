Instalação do Hydra

Hydra é uma ferramenta para ataques de força bruta em serviços de rede.

Linux (Debian/Ubuntu)

sudo apt update
sudo apt install hydra

Compilação manual

  1. Baixe o código fonte em github.com/vanhauser-thc/thc-hydra
  2. Extraia o arquivo baixado e entre na pasta:
    3. tar -xvzf hydra.tar.gz
cd hydra
  3. Compile e instale:
    4. ./configure
make
sudo make install

Windows

Versões pré-compiladas podem ser encontradas em sites de terceiros, mas recomenda-se usar ambientes Linux para maior compatibilidade.
