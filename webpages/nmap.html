O que é o Nmap?

O Nmap é uma ferramenta de código aberto usada para varredura e auditoria de redes.

Instalação

Instalação no Debian/Ubuntu:

sudo apt install nmap

Comando básico

Para escanear um IP específico:

nmap 192.168.1.1

Escaneamento de portas comuns

Para escanear as portas mais comuns:

nmap -F 192.168.1.1

Detecção de serviços e versões

Para identificar serviços e suas versões nas portas abertas:

nmap -sV 192.168.1.1

Escaneamento agressivo

Inclui detecção de SO, script scanning e traceroute:

nmap -A 192.168.1.1
