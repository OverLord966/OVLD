Instalação do Wireshark

O Wireshark é um analisador de pacotes de rede muito utilizado para capturar e inspecionar tráfego de rede.

Windows

  1. Acesse o site oficial: wireshark.org/download
  2. Faça o download do instalador para Windows.
  3. Execute o instalador e siga os passos na tela.
  4. Recomenda-se instalar o WinPcap ou Npcap para captura de pacotes.

Linux (Debian/Ubuntu)

sudo apt update
sudo apt install wireshark

Durante a instalação, você pode ser questionado sobre permitir que usuários não-root capturem pacotes. Configure conforme preferir.

macOS

Use o Homebrew para instalar:

brew install wireshark
