Instalação do Nessus

Nessus é um scanner de vulnerabilidades popular utilizado para auditoria de segurança.

Passos para instalação:

  1. Acesse https://www.tenable.com/products/nessus
  2. Escolha a versão apropriada para seu sistema operacional (Windows, Linux, macOS).
  3. Faça o download e execute o instalador.
  4. Siga as instruções na tela para configurar o Nessus.
  5. Após a instalação, acesse a interface web via https://localhost:8834.
