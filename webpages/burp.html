O que é o Burp Suite?

Burp Suite é uma plataforma integrada para testes de segurança em aplicações web. Ela permite interceptar, modificar e analisar o tráfego HTTP/S entre o navegador e o servidor, facilitando a identificação de vulnerabilidades.

Instalação

Para instalar o Burp Suite Community Edition no Linux (Debian/Ubuntu), siga os passos:

sudo apt update
sudo apt install openjdk-11-jre
wget https://portswigger.net/burp/releases/download?product=community&version=2024.3.1&type=jar -O burpsuite_community.jar
java -jar burpsuite_community.jar

Alternativamente, você pode baixar o instalador direto do site oficial aqui.

Configuração básica para interceptação

1. Abra o Burp Suite e selecione "Temporary Project".

2. Na aba "Proxy", certifique-se que o "Intercept" esteja ligado.

3. Configure o navegador para usar o proxy do Burp (por padrão, localhost na porta 8080).

4. Para evitar problemas com HTTPS, importe o certificado CA do Burp no navegador.

Exemplo básico de uso

Após configurar o proxy no navegador, navegue normalmente. O Burp Suite irá interceptar as requisições HTTP/S.

Você pode modificar os parâmetros e enviar as requisições para analisar as respostas e identificar possíveis vulnerabilidades.
