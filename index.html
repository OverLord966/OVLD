Sobre o Projeto

O OverSecurity é um projeto final desenvolvido no âmbito do curso de Cibersegurança. O objetivo é fornecer uma plataforma educativa que reúne tutoriais, ferramentas, notícias e testes interativos para reforçar o conhecimento e a consciência sobre segurança digital.

Tecnologias Utilizadas

  • Sistemas Operacionais: Kali Linux, Debian, Windows Server
  • Linguagens de programação: PHP, Ruby, Python, JavaScript
  • Ferramentas de varredura e análise: Nmap, Wireshark, Metasploit, Burp Suite, Nessus, OpenVAS, SQLmap
  • Ferramentas de cracking e força bruta: John the Ripper, Hydra
  • Simuladores e ambientes de rede: Cisco Packet Tracer, Sense
  • Editores e IDEs: Visual Studio Code
  • Plataformas de treinamento e CTF: Capture The Flag (CTF), TryHackMe, OverTheWire
